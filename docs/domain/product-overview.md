# プロダクト概要

## コンセプト

未来の目的と、今日の行動をつなぐ。

このアプリは単なる Todo アプリではありません。人生でやりたいことである Bucket List から逆算し、Project、Todo、そして「今やること」である Focus へ落とし込むことで、日々の行動に意味を持たせます。

小さな Todo が人生の目標につながっていることを可視化しつつ、日常の雑務も自然に扱えるよう、Todo は単独でも成立します。Bucket や Project への関連付けは任意です。

## 構造

```text
Bucket（人生の目的）
  ↓
Project（計画）
  ↓
Todo（行動）
  ↓
Focus（今やること）
```

## MVP

- Bucket 管理
- Project 管理
- Todo 管理
- Focus モード
- ローカル保存

## 主要機能

Home:
- 今日の Todo
- Focus 開始
- Bucket 進捗
- 最近完了した Todo

Todo:
- 作成、編集、削除、完了
- 並び替え、検索、フィルター
- タイトル、説明、タグ、優先度、締切、通知、Project 任意関連

Bucket:
- 作成、編集、削除
- 説明、ステータス、締切
- Project 一覧、Todo 一覧、進捗表示

Project:
- Bucket と Todo の中間概念
- Bucket なしでも作成可能
- Project 一覧画面は MVP では必須ではなく、Bucket 詳細や Todo 詳細から辿れる形を優先

Focus:
- 現在やるべき Todo を 1 件だけ表示
- 関連 Bucket がある場合のみ、その Todo がどの未来につながるかを表示
- 考える時間を減らし、行動する時間を増やす体験を重視

## ロードマップ

Phase 1:
- Todo
- Bucket
- Project
- Focus
- ローカル保存

Phase 2:
- カレンダー
- タグ
- 通知
- 統計
- 振り返り

Phase 3:
- クラウド同期
- ログイン
- Web 対応
- AI による Todo 分解・優先順位提案
- ウィジェット

## 判断基準

既存の Todo アプリが「何をやるか」を管理するのに対し、このアプリは「なぜそれをやるのか」「その行動が未来のどこにつながっているのか」を自然に見せることを目指します。

機能追加や仕様変更では、Focus 体験を中心に据えます。
