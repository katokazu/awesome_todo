# Focus 仕様

Focus はこのアプリの中心体験です。

## 目的

ユーザーがリストを眺めて迷う時間を減らし、今やる Todo に集中できるようにします。

## 表示

- Todo を 1 件だけ表示する
- Todo に関連する Bucket がある場合、その Bucket とのつながりを表示する
- Bucket がない Todo では、未来への接続表示を省略する

表示例:

```text
今やること

ホテルを予約する

この Todo は
北海道一周
につながっています。
```

## フロー

```text
Home
  ↓
Focus 開始
  ↓
Todo 表示
  ↓
完了
  ↓
次の Todo
  ↓
ホーム
```

## 操作

- 開始
- 後で
- 完了
- ホームへ戻る
